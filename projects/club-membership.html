FurLodge:The Airbnb for Pet Boarding

  • Tech StackNodeJS, npm, ExpressJS, express-session, bcryptjs, passport, passport-local strategy, sequelize, mysql2, express-handlebars, Foundation CSS framework, jQuery and heroku.
  • Github URL: Project Link

FurLodge is an innovative web application designed to connect pet owners with loving hosts who are eager to care for pets while their owners are away. Inspired by the Airbnb model, this marketplace ensures that our beloved furry friends find real homes where they can stay, play, and receive the affection they deserve. The platform features a mobile-first, clean, and polished user interface that seamlessly adapts to multiple screen sizes, providing an optimal experience on any device. Adherence to ARIA accessibility rules and guidelines ensures inclusivity for all users.
