  • Tech Stack: Python, Java, TensorFlow, Keras, OpenCV, Flask, Spring Boot, Firebase
  • Github URL: Project Link

This project focuses on creating an application that allows specially-abled people to interact with each other, irrespective of their disabilities, by leveraging vision-based solutions. The application provides users with functionalities such as messaging, speech-to-text, and hand gesture-to-text conversion.

A comprehensive dataset of the Indian Sign Language (ISL) was created, containing 87,500 images for letters A-Z and numbers 1-9. To handle this large dataset, various object detection and image classification algorithms were implemented.

The final algorithm, which combined YoloV4 and Resnet152V2, achieved an impressive accuracy of 96.12% in gesture detection. This project involved extensive data preprocessing, cleaning, and feature engineering, demonstrating the ability to manage large datasets and develop robust machine learning models.

Additionally, we authored and published a paper on this project titled “Application for specially-abled people using hand gesture recognition” at the 6th International Conference for Convergence in Technology (I2CT). The paper is accessible through the DOI: 10.1109/I2CT51068.2021.9417924 (May 2021).
