JobTrek - A Job Tracking Web App

Project Link Backend

JobTrek is a web-based application that helps job seekers manage and track their job applications effortlessly. The frontend is built with ReactJS, offering a clean, user-friendly interface styled with TailwindCSS and utilizing Redux Toolkit for efficient state management. Data fetching is handled using Axios, ensuring seamless updates. The backend is developed using NodeJS and ExpressJS, providing robust server-side functionality and handling API requests efficiently. Media uploads are managed by Cloudinary, allowing for secure and scalable storage of user-uploaded files. By organizing applications and statuses in one accessible platform, JobTrek simplifies the job search process and eliminates the need for spreadsheets or email searches.
